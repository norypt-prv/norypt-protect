Norypt Protect

Local-only Android security. Lock, wipe, and harden your device — with zero network access, zero logs, and zero cloud dependency.

GPL-3.0 No Internet Android 13+

What it does

Panic

5× Power Button

Press the power button five times to instantly lock the screen or trigger a full device wipe.

Panic

Dead-Man Switch

If the device is not checked in within a set interval, it wipes automatically.

Panic

Low-Battery Wipe

Wipe when the battery drops below a threshold and the device has not been unlocked.

Hardening

Device Admin / Owner

Activate as Device Admin for lock enforcement. Escalate to Device Owner via ADB for full control.

Hardening

Work-Profile Wipe

Wipe only the work profile, leaving the personal profile intact when running as Device Owner.

Norypt

Emergency SOS Disable

Auto-disables Emergency SOS on Device Owner promotion to prevent accidental calls.

Norypt

Hide from Launcher

Toggle the launcher alias to hide the app icon from the app drawer.

Norypt

Notification Monitor

Detect suspicious notification patterns and respond with configurable actions.

Verify the APK

Every release APK is signed and has a published SHA-256 checksum. Verify before installing:

sha256sum norypt-protect-1.0.0.apk

Latest release SHA-256:

LATEST_SHA256_PLACEHOLDER

The checksum is also published in the GitHub Release and in the attached .sha256 file. Builds are reproducible — see reproducible-build.md.
